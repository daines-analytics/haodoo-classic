伏羲氏《馬雲的話》2014/4/11

馬雲在中國科技界非常成功和有名，他在辭去ＣＥＯ職位後，留下了對創業和科技工作者的臨別贈言。真心誠意，令人感動，可以說句句是肺腑之言。讀後更令人懷念他的離去。所以我把他的留言，先抄在下面。我的感言留在後面。大家可以略去或不讀我的感言，但「馬雲的話」一定得讀。尤其是你還在工作，更尤其是你在擔任領導，更更尤其是你想創業的話。

馬雲的話

一、我最遺憾的錯誤

01年，我犯了一個錯誤，我告訴我的18位共同創業同仁，他們只能做小組經理，而所有的副總裁都得從外面聘請。現在十年過去了，我從外面聘請的人才都走了，而我之前曾懷疑過其能力的人都成了副總或董事。我相信兩個信條：態度比能力重要，選擇同樣也比能力重要！

二、不能統一人的思想，但可以統一人的目標

千萬不要相信你能統一人的思想，那是不可能的。30％的人永遠不可能相信你，不要讓你的同事為你幹活，而讓他們為我們的共同目標幹活。團結在一個共同的目標下，要比團結在一個人周圍容易的多。

三、提醒：細節好的人格局一般都差
  1. 有人覺得我牛，6分鐘說服了孫正義，其實是他說服了我。見孫正義之前，我在矽谷至少被拒絕了40次。
  2. 做企業贏在細節，輸在格局。
  3. 格局，「格」是人格，「局」是胸懷，細節好的人格局一般都差，格局好的人從來不重細節，兩個都幹好，那叫太有才！
四、領導比員工多什麼？

領導永遠不要跟下屬比技能，下屬肯定比你強；如果不比你強，說明你請錯人了。
  1. 要比眼光：比他看得遠；
  2. 要比胸懷：領導的胸懷是委屈撐大的，要能容人所不容；
  3. 要比實力：抗失敗的能力比他強；一個優秀的領導人的素質就是眼光、胸懷和實力。
五、中國商人千萬別在「紅道」上混
  1. 錢和權兩個東西是絕對不要碰在一起，當了官永遠不要想有錢，當了商人千萬別想權；
  2. 錢和權這兩個東西碰在一起就是炸藥和雷管碰在一起，必然要爆炸；
  3. 胡雪岩的悲哀就在於他是紅頂商人；
  4. 中國商人千萬別在「紅道」上混。
六、年輕人必須思考的四大問題
  1. 什麼是失敗？放棄就是最大的失敗。
  2. 什麼叫堅強？經歷許多磨難、委屈、不爽，你才知道什麼叫堅強。
  3. 你的職責是什麼？比別人多勤奮一點、多努力一點、多一點理想，這就是你的職責。
  4. 傻瓜用嘴講話，聰明人用腦袋講話，智者用心講話。
七、人生在世在做人，不是做事

我跟自己講我們到這個世界上不是來工作的，我們是來享受人生的，是來做人不是做事。如果一輩子都做事的話，忘了做人，將來一定會後悔。不管事業多成功、多偉大、多了不起，我們到這個世界就是享受經歷這個人生的體驗。忙著做事一定會後悔。

八、高手的競爭論
  1. 一定要爭得你死我活的商戰，是最愚蠢的。
  2. 眼睛中全是敵人，外面就全是敵人。
  3. 競爭的時候不要帶仇恨，帶仇恨一定失敗。
  4. 競爭樂趣就像下棋一樣，你輸了，我們再來過，兩個棋手不能打架。
  5. 真正做企業是沒有仇人的，心中無敵，天下無敵。
九、別把抱怨當習慣

人是退化最嚴重的動物。跟獸比人很「弱肢」，和狗比人很「聞盲」，但人類「進化」了抱怨。偶爾為之無大礙，但當抱怨成習慣，就如喝海水，喝的越多渴得越厲害。最後發現，走在成功路上的，都是些不抱怨的「傻子們」。世界不會記得你說了什麼，但一定不會忘記你做了什麼！

十、給初創企業者的忠告
  1. 大家看不清的機會，才是真正的機會。
  2. 讓員工笑著幹活。
  3. 客戶第一、員工第二、股東第三。
  4. 搶在變化之前先變。
  5. 忘掉money，忘掉賺錢。
  6. 小聰明不如傻堅持。
  7. 心態決定姿態，姿態決定狀態。
十一、談創業
  1. 一個好的東西往往是說不清楚的，說得清楚的往往不是好東西！
  2. 創業要找最合適的人，不一定要找最成功的人。
  3. 這世界最不可靠的東西就是關係。
  4. 免費是世界上最昂貴的東西。
  5. 今天很殘酷，明天更殘酷，後天很美好。
十二、「四不」創業智慧
  1. 創業最怕就是看不見，看不起，看不懂，跟不上；
  2. 看不見對手在哪裡，看不起對手，看不懂對手為什麼變得那麼強，然後就跟不上了；
  3. 即使對手很弱小，也一定要把對方看的很強大，即使對手很強大，也不一定要把自己看的很弱小。
十三、當你決定要創業時，便意味著：
  1. 沒有了穩定的收入；
  2. 沒有了請假的權利；
  3. 沒有了得紅包的機會。然而卻更意味著：
    1. 收入不再受限制；
    2. 時間運用更有效；
    3. 手心向下不求人。
想法若不同，結果便不同；選擇不一樣，生活才變樣。

十四、談機會

如果一個方案有90％的人說「好」的話，我一定要把它扔到垃圾桶裡去。因為這麼多人說好的方案，必然有很多人在做了，機會肯定不會是我們的了。

我的感言

凡是過去或現在從事科技業的人，無論擔任過什麼樣的職位，多多少少都會想到去創業。一方面是機會難得，有這個條件和機會，再方面是在我們中國人的血液裡面，無論中港台，無論老中青，淌著的都是創業的血。實際一點，白一點的說，就是個個都想當老板。我自己也不曾例外，況且還曾經做過幫人家創業的工作。所以更應該貢獻一些心得。

二十年前，我在加州的矽谷，現在叫硅谷吧，遇到的頭一個例子，就是那位堅持要創業的仁兄。有天晚上，仁嫂到我家來說：「蔡大哥，我拜託你無論如何，都勸勸我老公，我實在搞怕了。這已經是他第三次開公司了。傾家蕩產不說，每天暗無天日，不見人影的幹下去，要到那一天才能出頭呀？不要搞到公司還沒有開成，人已經先掛了。」。對某些男人來說，這世界上還有比去賭埸，找小三更令人興奮著迷的事，那就非創業做老板莫屬了。

玩古董的人都曉得，有時候在鑑賞一件古玩的時候，會有看走眼或誤估一件寶物的價值的事情發生。行內的人都稱這種事叫「打眼」。做我們這一行，創業投資的，「打眼」更是常有的事。舉一個切身之痛的例子。

我有一位中學大學的同學，因為自己忙於發財而且也已經發財，問我他的獨子將來做什麼最好。我在詳細問清了狀況和背景後，建議他最好還是按照他們家的傳統，自己出來開公司做老板。他非常自信的問我，有那一個最成功和現成的例子。我說那當然是蘋果的賈伯斯了。他說：「行！就這麼辦！」把他的獨子，從落杉磯加大轉到史丹福大學電機系。畢業後果然按照我們設計的藍圖，和兩個老美同班同學開起了公司。還娶了個同班韓國女同學做老婆，那倒不在我們規劃之內。

過了不久，同學夫婦倆，就帶來了他們寶貝兒子的企劃書來找我們公司投資。扔下企劃書，三言兩語，夫婦倆就閃人，忙著去發其他的財了。我仔細一看傻眼了。原來搞了這麼久，他們發明的是電子指南針，說已經賣了頭批貨給美軍在中東和北非做測試之用。我不敢大意，還拿去給我老板幫忙評估一下。結果被他罵到臭頭，說：「全世界除了北非和中東之外，就只剩下新疆的戈壁沙漠可以用得上，請問還能賣給誰？」

只好硬了頭皮跟老同學說抱歉。多年過去，去了趟賭城，到他家別墅做客。茶餘飯後才知道他兒子發了大財。就是靠那個該死的電子指南針。原來他們繼續研發後，賣給美國三大汽車廠之一。你現在車上照後鏡上就可能有他們的產品。那個小小的，在左或右下角告訴你，現在是朝東南西北那個方向開。至於他們發到什麼程度，我那好意思問。主人說他們的小孫女現在在邁阿密，請了個職業教練教她打網球。那其他的就更別問了。這不就是「打眼」還是什麼！

已經舉了兩個極端的例子，那麼一般的情形又如何呢？中文有句諺語說得好：「十賭九輸！」大家一定舉手發問：「創業為什麼是賭博？又為什麼只有10％成功的機會？」

先回答頭一個問題。創業或者任何事情成功的話，一定要符合「天時，地利，人和」的基本條件。

「天時」就是timing，時機對不對？早發了，買家不識貨，沒人欣賞，賣不出去。晚發了me too的產品，狗都嫌。

「地利」是指你在什麼樣的環境做起來的。譬如說同樣是全球最尖端的高科技研發，在加州的硅谷，在北京的中關村，在上海的張江和在台灣的南港，就會有完全不一樣的成長和命運。

「人和」更是關鍵中的關鍵。一起打天下的時候，大哥二哥麻子哥，幹起活來，誰都搶著幹。分攤研發費用時也都是爭先恐後，唯恐少了以後的分紅。等到東西賣出去了，個個都像翻書一樣的翻臉。請問如何維持人和？總而言之，閞公司和賭博沒兩樣。人人都興奮，個個沒把握。

再回答第二個問題「為什麼只有10％成功的機會？」因為除了上述的三個不利的因素外，還有其他幾個不利於成功的因素。頭一個就是投資者肯不肯投錢？像我過去工作的公司，每年都會收到近兩千份的企劃書，而真正挑出來肯出錢的不過廿個。百分之一的比例也未免太小了一點。再一個因素就是從研發成功到大量生產有一段距離。再一個因素是從量產到量售又是一個挑戰。所以說「百年老店」這四個字，不知包含了多少人，多少代，心血的結晶。

因為大陸的市場已經開放和逐漸自由化，所以創業的人愈來愈多，馬雲提到的幾個具有「中國特色」的幾點更是值得關注。頭一個是「關係」已經愈來愈式微。不能說不重要，但是千萬不要自欺欺人。自己想要喝兩杯，自己想要人來瘋，就別找藉口和拖人下海。何況你現在最寶貴的是「龍體」和「時間」，一點點都不容有絲毫的閃失。

「免費是世界上最昂貴的東西」。沒錯！貪便宜是人類的天性。在創業唯艱的初期，尤其是創業基金制度不是十分健全和普遍的中港台，創業者要面臨資金短缺的挑戰是空前的。在你創業的過程中，有人無端的幫你籌錢，有人沒事的提供你無償的週轉，你都千萬要小心。在這個現實的社會中，沒有白吃的午餐。沒有人會無緣無故的對你好。

「官商勾結」是人人都知道要避免、要小心的應付。如果你說「我才不理呢！」那是你太天真了。就像蔣介石的兒子，蔣經國說過的：「為人民服務就像是去水溝排除污泥，怕髒的別下來！」。同樣的，走上這條創業不歸路，只有勇往直前，成功為止。把捏分寸的原則是：犯法的事一定不做；其他的，儘量發揮你的想像力、忍耐力，和親和力。「關係」不是不好使。但就像是向朋友借錢一樣。先想清楚要怎麼還？還不還得了？再借不難。

「客戶第一，員工第二，股東第三」。我們會很自然的排成「股東第一，員工第二，客戶第三」。因為看不到客戶，因為員工和我們一起打天下，因為股東是我們的衣食父母，自然會有這個排序。馬雲就是馬雲，他看得遠，也許是他的慘痛經驗吧。其實仔細想想也對。不把客戶放在第一位，他為什麼要買你的東西？不把員工放在心上，他為什麼要替你賣命？至於股東，在股東們的心目中，他們所關心的，只是何時可以把他們投資的錢加上賺到的拿回來而已。

「人生在世是在做人，不是在做事。」這是馬雲最了不起的地方。他不是大學教授，他不是市委書記，而他每天接觸到的都是爾虞我詐，極端現實的投機客。他怎麼會說樂在工作？又怎麼會說我們是來享受人生的呢？因為他一定經過人世間最醜惡的一面，因為他一定走過那最黑暗的道路。所以他才會珍惜，所以他才會感嘆的說要大家好好做人，要大家好好享受人生。

這些都是馬雲最寶貴的經驗。我們誠摯的感謝他肯和我們一同分享。也祝福他未來的路，一定會平安愉快的走下去。
